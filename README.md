# Diabetes Risk Analysis

## Overview
This project analyzes patient health data to identify key factors associated with diabetes risk. The workflow demonstrates **data cleaning, transformation, and visualization** using **SQL and Power BI**, highlighting how raw data can be turned into actionable insights for healthcare decision-making.

---

## Project Objectives
- Clean and preprocess the dataset using SQL for accurate analysis.  
- Identify major risk factors contributing to diabetes, such as family history, age, and lifestyle.  
- Visualize trends and patterns using an interactive Power BI dashboard.

---

## Tools & Technologies
- **SQL (SSMS):** Data cleaning, transformation, and querying.  
- **Power BI:** Interactive dashboards and visual insights.
