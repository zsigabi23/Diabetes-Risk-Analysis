# Diabetes Risk Analysis

**Description:**  
This project analyzes patient data to identify factors associated with diabetes risk. It includes data cleaning, exploratory analysis, and visualization using Excel, SQL, and Power BI.

**Tools & Technologies:**  
- SQL (SSMS)  
- Power BI  

**Key Achievements / Insights:**  
- Cleaned and standardized the dataset, converting categorical values into binary format.  
- Identified major risk factors such as family history, age, and lifestyle habits.  
- Built an interactive Power BI dashboard to visualize trends and support healthcare decisions.

**Project Files:**  
- `dataset.csv` – The dataset used for analysis  
- `data_cleaning.sql` – SQL scripts used for cleaning and analysis  
- `dashboard.pbix` – Power BI dashboard  
- `analysis.xlsx` – Excel analysis and transformations

**Usage:**  
Open the files in the corresponding software (SSMS, Power BI) to explore the analysis and visualizations.
